📝 Notes App

A simple and responsive Notes Application built using React.js that allows users to add, edit, and delete notes. The application uses React Hooks (useState, useRef) for state management and input handling, and Local Storage to persist notes across browser sessions.

🚀 Features

✅ Add new notes
✅ Edit existing notes
✅ Delete notes
✅ Auto-focus input field using useRef
✅ Persistent data storage using localStorage
✅ Clean and responsive UI
✅ Built with React Functional Components and Hooks

🛠️ Technologies Used
React.js
JavaScript (ES6+)
HTML5
CSS3
React Hooks (useState, useRef)
Browser Local Storage
